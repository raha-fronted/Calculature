# Calculature

A modern and responsive calculator built using HTML, CSS, and Vanilla JavaScript.

## 🚀 About the Project

This project was created to practice DOM manipulation, event handling, and implementing core JavaScript logic.  
The main focus was building a clean user interface and writing structured, readable code.

## 🛠 Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6)

## 📌 Purpose

The goal of this project is to strengthen frontend development skills and improve understanding of fundamental JavaScript concepts.
